# Política de Privacidade

A extensão **Nome do Atendente para WhatsApp** não coleta, transmite, vende ou compartilha dados pessoais dos usuários.

A extensão armazena localmente no navegador apenas o nome do atendente configurado pelo próprio usuário. Esse dado é salvo usando o armazenamento local do Firefox e é utilizado exclusivamente para inserir automaticamente a identificação do atendente nas mensagens enviadas pelo WhatsApp Web.

A extensão não envia informações para servidores externos, não utiliza serviços de análise, não coleta histórico de navegação, não coleta mensagens, não coleta contatos e não acessa dados fora do WhatsApp Web.

O usuário pode alterar o nome do atendente a qualquer momento pela interface da extensão. Ao remover a extensão, os dados locais associados a ela podem ser removidos pelo próprio navegador.

## Permissões utilizadas

### storage

Usada para salvar localmente o nome do atendente configurado pelo usuário.

### https://web.whatsapp.com/*

Necessária para inserir o nome do atendente no campo de mensagem e legenda de arquivos no WhatsApp Web.
